The example files are provided for you to get some hands-on experience while you follow along with the video. 
This module includes an Access .mdb format database.

Extract the files from the zip archive and place them in a convenient location on your local hard drive.

It is recommended you place the .mdb file in a new folder off the root of your drive, 
as you will be required to enter the entire drive and folder path during the follow-along video – 
keeping this path short will make typing easier.
The video uses the location: C:\Data
